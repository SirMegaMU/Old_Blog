### 转义字符

| 转义字符 | 含义                |
| -------- | ------------------- |
| `\a`     | 警报                |
| `\b`     | 退格符              |
| `\f`     | 换页符              |
| `\n`     | 换行符              |
| `\r`     | 回车符              |
| `\t`     | 水平制表符          |
| `\v`     | 垂直制表符          |
| `\\`     | 反斜杠              |
| `\'`     | 单引号              |
| `\"`     | 双引号              |
| `\?`     | 问号                |
| `\0`     | 空字符              |
| `\ddd`   | 1~3位任意八进制字符 |
| `\xhh`   | 任意十六进制字符    |
